== dd

[source,bash]
.ディスクの完全消去
----
# 対象のデバイスを特定
fdisk -l
# 消去(of に対象のデバイスを指定)
dd if=/dev/zero of=/dev/sdX bs=4096 status=progress
# No space left on device(デバイス上に空きスペースがありません) で完了

----
